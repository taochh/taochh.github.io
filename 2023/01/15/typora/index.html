1. 快捷键

  1. Ctrl + O 打开文件 O=open
  2. Ctrl + P 快速打开最近的文件
  3. Ctrl + W 关闭文件
  4. Ctrl + Tab 应用内窗口切换
  5. : + 任意单词 各种微表情
  6. [TOC] 设置文档目录，或者直接右键–>插入–>目录
  7. Fn + ← 或 Fn+→ 跳到行开头或结尾,同时按住Shift 即选中
  8. Ctrl + ← 或 Ctrl + → 跳过一个单词，同时按住Shift 即选中

  1. Ctrl + / 源代码模式

  2. Ctrl + Shift + L 显示隐藏侧边栏

  3. F8 专注模式

  4. F9 打字机模式

  1. Ctrl + Shift + C 复制为markdown格式

  2. Ctrl + L 选中当前行

  3. Ctrl + D 选中当前词

  4. Ctrl + Home 跳转至文档开头

  5. Ctrl + End 跳转至文档末尾

部分markdown语法用快捷键更方便

  1. Ctrl + 1 一级标题，1-6分别为1-6级标题

  2. Ctrl + + 提升标题级别

  3. Ctrl + - 降低标题级别

  4. Ctrl + Shift + K 插入代码块

  5. Ctrl + Shift + M 插入公式快

  6. Ctrl + Shift + X 任务列表

  1. Ctrl + T 插入表格
  2. Ctrl + Enter 表格下方插入一行
  3. Alt + ↑ 将表格该行上移，↓ 将该行下移
  4. Alt + ← 将表格该列左移，→ 将该列右移
  5. Ctrl + Shift + Backspace 删除表格中某一行
  1. Ctrl + B 加粗
  2. Ctrl + K 超链接

2. markdown语法

Typora 的 Markdown 语法 - Typora Support (typoraio.cn)

部分语法比typora快捷键更方便

  1. > 引用

  2. 1. 2. 3. 有序列表

  3. - 无序列表

  4. Tab 列表增加缩进

  5. Shift + Tab 列表减少缩进

  6. ^^ 上标，在两个^之间输入需要上标的内容

  7. ~~ 下标，在两个~之间输入需要下标的内容

  8. == == 高亮，在两个==之间输入需要高亮的内容

  9. --- 水平分割线

  10. <br> 换行不分段

数学公式中的部分语法

  1. a^b 上标，公式中显示为a^b^

  2. a_2 下标，公式中显示为a2

  3. \frac{a}{b} 分数，两个大括号中输入分子和分母，该例显示为：
    $$
    \frac{a}{b}
    $$
